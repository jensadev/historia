DOM - Document object model

HTML mönster, element

            <tag attribut>value</tag>
            <h1>Detta är en rubrik</h1>
            <nti>Detta är en tag som inte finns</nti>

Här använder jag nti taggen, den renderar fortfarande

Designtanke bakom detta. Vi kan alltid lägga till element och det kommer fortfarande visas på gamla läsare.

Vi kan använda det som fallback. Innehållet i taggar som inte stöds visas alltid.

            <canvas>Your browser does not support the HTML5 canvas tag.</canvas>

Gör språket robust, det har funnits i över 20 år på samma spec
